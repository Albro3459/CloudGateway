# These are example CloudFlare DNS records for your domain.
# My domain is gocloudlaunch.com, so that is what I will use for this example
# These records are what I use for the following:
# * Hosting the frontend React web app on GitHub pages
# * Routing to the APIs on each server (2 servers: San Jose and Chicago)
# * Routing to WireGuard on each server (2 servers: San Jose and Chicago)
# * Emails with both AWS SES (successful client creation and removal) and Firebase (reset password emails)

;; SOA Record
gocloudlaunch.com	3600	IN	SOA	marek.ns.cloudflare.com. dns.cloudflare.com. 2053314179 10000 2400 604800 3600

;; NS Records
gocloudlaunch.com.	86400	IN	NS	marek.ns.cloudflare.com.
gocloudlaunch.com.	86400	IN	NS	maxine.ns.cloudflare.com.

;; A Records
